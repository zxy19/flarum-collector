# Collector

![License](https://img.shields.io/badge/license-MIT-blue.svg) [![Latest Stable Version](https://img.shields.io/packagist/v/xypp/collector.svg)](https://packagist.org/packages/xypp/collector) [![Total Downloads](https://img.shields.io/packagist/dt/xypp/collector.svg)](https://packagist.org/packages/xypp/collector)

A [Flarum](http://flarum.org) extension. Data Collector collects all kinds of data for other extension to use.

## Installation

Install with composer:

```sh
composer require xypp/collector:"*"
```

## Updating

```sh
composer update xypp/collector:"*"
php flarum migrate
php flarum cache:clear
```
